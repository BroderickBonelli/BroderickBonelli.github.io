Recent Work

Sector Performance Model

Scrapes price data for sector ETF'S and plots the rolling 1w, 1mo, and 3mo performance. This helps gauge sector momentum at a glance.

Sector Fundamentals Model

Scrapes fundamental metrics and ratios for all stocks in a specific sector, and ranks them visually. This is a quick way to find the strongest and weakest names in a sector from a fundamental perspective.

Dividend Growth Charts

Scrapes maximum dividend data for a specific stock and charts the annual dividends and dividend growth rates over time. This is a useful tool for dividend growth investors to quickly see how a company has grown its dividends over the years.

BTC Address Analysis

Analysis on % Change in Unique BTC Addresses as a leading indicator of BTC Price. Read data in, clean and format. Merge two datasets together based on common date values. Plot each separately, and then together with two y-axis'. Highlight bullish (green) and bearish (red) divergences on twin axis chart.

ETH Address Analysis

Study of the growth in the Number of Unique ETH Addresses. Read data downloaded from Etherscan and Coindesk into Jupyter Notebook, clean, format and chart.

Recession Timing Model

Read historical VIX and ISM Manufacturing PMI data into a dataframe, and chart. Highlight dates on x-axis where the yield curve is inverted (ie: 10y yield is lower than 2y yield). The aim of this model was to see if Yield Curve inversion and the PMI had any indication of a looming vol event. My conclusion is that this is true. If the Yield Curve Inverts and the ISM is below 50, it is a reliable leading indicator of a volatility event where VIX spikes above 45.

NFL Receiving Stats 2021-22 - Tableau

This is a study of all NFL players with more than 500 receiving yards during the 2021-22 season. Here I use python and Tableau to analyze and visualize the data.

Dune Analytics Aragon Growth Analysis

Use Postgre SQL to query and visualize Aragon DAO data with Dune Analytics. Measure things such as # of new DAO's created monthly, price of the native token ANT, # of votes amongst all DAO's, etc.

Token Supply & Price Regression Analysis

Use BeautifulSoup to web scrape Price and Circulating Token Supply data. Clean, format and compile data into a pandas Dataframe. Create a log, log, scatterplot with MatPlotLib and fit a regression line to it using scipy. This demonstrates the clear relationship between token supply and price: the greater the supply, the lower the price.
